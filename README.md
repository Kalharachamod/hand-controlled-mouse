# Hand-Controlled Mouse

A Python application that allows you to control your computer mouse using hand gestures detected through your webcam.

## Requirements

- Python 3.11 (MediaPipe is not compatible with Python 3.13)
- OpenCV
- MediaPipe
- PyAutoGUI
- NumPy

## Setup

1. Create a virtual environment with Python 3.11:
   ```
   py -3.11 -m venv hand_mouse_env
   ```

2. Activate the virtual environment:
   ```
   hand_mouse_env\Scripts\activate
   ```

3. Install the required packages:
   ```
   pip install opencv-python mediapipe pyautogui numpy
   ```

## Usage

Run the application:
```
python main.py
```

## How to Use

1. Show your hand to the webcam
2. Move your index finger to control the mouse cursor
3. Bring your thumb and index finger close together to simulate a mouse click
4. Press ESC to exit the application
